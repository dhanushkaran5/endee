# 🌳 Smart Tree Plantation & Carbon Reduction Calculator (AI-Based)

## 📌 Overview
This project calculates carbon reduction based on trees planted and provides intelligent recommendations using simple AI logic.

It helps users choose the best trees based on budget, CO₂ absorption, and environmental impact.

---

## 🎯 Objective
To build a practical AI-based system that demonstrates:

- Carbon reduction calculation
- Hybrid search for trees
- Smart recommendation system
- Environmental impact analysis

---

## 🧠 Key Concepts Demonstrated

- Rule-Based AI System
- Hybrid Search Algorithm
- Recommendation & Matching Logic
- Data Processing using CSV
- User Input Handling

---

## ⚙️ System Architecture

1. User enters:
   - Tree type / search input
   - Budget
   - CO₂ goal

2. System processes input

3. Hybrid search filters trees

4. AI logic evaluates best match

5. Output is displayed to the user

---

## 🤖 AI Logic Used

Carbon Reduction:
CO₂ = Number of Trees × CO₂ absorption per tree

Recommendation Logic:
- If price ≤ budget
- If CO₂ ≥ goal
→ Suggest best trees
